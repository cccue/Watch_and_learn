The first version of this watch comparison prototype I built early 2016 while at Insight Data Science in NYC. However I keep on adding improvements when interesting ideas come up that have potential to enhance this tool. The goal is to compare watches by similarity using a color-based and a design-based schemes. For the design-based part I have used a convolutional neural network, therefore expect the algorithm to take about 15s to generate the results (using the caffe library from Berkeley running on CPU mode). As of now the prototype will NOT work on any random image of a watch but was built to work on a seller database. For more info and example images to query the algorithms visit : Sample pictures Only jpg format supported so far. Want to know more?
